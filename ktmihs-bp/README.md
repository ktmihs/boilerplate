# ktmihs-bp

ktmihs's boilerplate!

# Install

```bash
npx ktmihs-bp [PACKAGE_NAME] [PROJECT_NAME]

```

### example

```bash
npx ktmihs-bp react-webpack-ts new-react-app
```
